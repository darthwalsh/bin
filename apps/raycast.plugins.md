## snippets from disk
Raycast's built-in snippet feature is designed for manually created or imported content (via JSON file). It does not natively support reading snippet content directly from a file on your disk.

To make and edit and persist to "dotfiles":
1. Raycast > Snippets search > Edit
2. When done, Export and save over i.e. [[ai summary.json]]

However, another methods allow you to use file content for snippets:
- The [Snippet Surfer extension](https://www.raycast.com/astronight/snippetsurfer) for Raycast is specifically designed to read snippets from a folder of Markdown or YAML files on your disk.
